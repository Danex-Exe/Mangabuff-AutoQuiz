## Mangabuff Helper

`Mangabuff Helper` — это userscript для Tampermonkey под `mangabuff.ru`, который объединяет автоответы в квизе, автоскролл глав, лайки и комментарии к главам, а также автодобычу в одной панели управления.

## Файлы

- `../../mangabuff_helper.js` — единственный рабочий userscript в репозитории.
- `../../mangabuff_autoquiz.js` — устаревшая совместимая заглушка. Логика autoquiz перенесена в `mangabuff_helper.js`, отдельно этот файл больше не нужен.

## Возможности

- `AutoQuiz`
  Запускает квиз на поддерживаемых страницах, берёт `correct_text` из ответа API и продолжает отвечать, пока квиз не закончится.
- `AutoScroll`
  Работает на страницах чтения, прокручивает главу, ищет кнопку следующей главы в футере, открывает следующую главу и сам выключается, когда следующей главы больше нет.
- `Auto Like`
  Пока включён автоскролл, скрипт отправляет один запрос `/favourite` на каждую главу, используя id из кнопки лайка.
- `Auto Comment`
  Пока включён автоскролл, скрипт пытается отправлять один случайный короткий комментарий каждую 2 или 4 главу.
- `AutoMine`
  Отправляет `POST /mine/hit` через XHR и выключается при ошибке запроса или достижении дневного лимита.
- `Панель управления`
  Показывает checkbox-переключатели, текущие статусы модулей и модальное окно с настройкой силы и интервала прокрутки.

## Установка

1. Установите [Tampermonkey](https://www.tampermonkey.net/).
2. Создайте новый userscript.
3. Скопируйте в него содержимое [`../../mangabuff_helper.js`](../../mangabuff_helper.js).
4. Сохраните скрипт.
5. Откройте `mangabuff.ru` и используйте кнопку `MB` в левом нижнем углу.

## Использование

1. Откройте любую страницу на `mangabuff.ru`.
2. Нажмите на плавающую кнопку `MB`.
3. Включите нужные модули.
4. Для страниц глав откройте настройки автоскролла и настройте:
   - силу прокрутки в пикселях за шаг;
   - интервал между шагами прокрутки в миллисекундах.

## Сценарий работы в ридере

На поддерживаемых страницах глав скрипт:

1. Определяет id текущей главы по `.reader-menu__item--like`.
2. Отправляет один лайк для этой главы.
3. Считает главы и случайно решает, нужно ли писать комментарий на 2-й или 4-й главе.
4. Прокручивает страницу до футера.
5. Переходит по ссылке `След. глава`, если она есть.
6. Выключает автоскролл, если ссылки на следующую главу больше нет.

## Примечания

- Скрипт зависит от текущей структуры DOM и поведения API сайта `mangabuff.ru`.
- Если сайт изменит классы, payload запросов или разметку футера, часть функций может перестать работать до обновления скрипта.
- Отправка комментариев всё ещё может блокироваться антиспамом сайта.
- Автоматизация действий на сайте может нарушать его правила. Используйте скрипт на свой риск.
