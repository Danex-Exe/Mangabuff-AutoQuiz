## Mangabuff-helper

`Mangabuff-helper` — userscript для Tampermonkey под `mangabuff.ru`, который объединяет автоответы в квизе, автоматизацию ридера, автодобычу, встроенную панель чата и локальные всплывающие уведомления в одной панели управления.

## Версия

Текущая версия userscript: `2.1.0`

## Файлы

- `../../mangabuff_helper.js` — единственный рабочий userscript в репозитории.
- `../../mangabuff_autoquiz.js` — устаревшая совместимая заглушка. Основная логика перенесена в `mangabuff_helper.js`.

## Модули ридера

- `AutoScroll`
  Прокручивает страницу главы с настраиваемой силой и интервалом.
- `Auto Chapter`
  Переключает на следующую главу, когда достигается конец страницы.
- `Auto Likes`
  Отправляет один `/favourite` запрос для каждой новой главы и показывает всплывающее уведомление.
- `Auto Comments`
  Отправляет один короткий комментарий раз в `N` глав и показывает всплывающее уведомление.

## Поведение

- Когда автоскролл переключает на новую главу, скрипт сохраняет его состояние и повторяет запуск на новой странице, пока разметка ридера не станет доступна.
- Успешные ответы квиза логируются в консоль браузера.
- События автолайков и автокомментов показываются как локальные popup-уведомления.
- Скрипт не запускается внутри встроенных iframe, поэтому панель чата не создаёт вложенные копии helper-скрипта.

## Остальные модули

- `AutoQuiz`
  Запускает квиз, берёт `correct_text` из ответа API и продолжает отвечать, пока квиз активен.
- `AutoMine`
  Отправляет `POST /mine/hit` через XHR. Если endpoint возвращает `403`, скрипт пытается использовать видимую кнопку шахты.
- `Built-in Chat`
  Открывает правую выезжающую панель со встроенной страницей чата внутри `iframe`.

## Установка

1. Установите [Tampermonkey](https://www.tampermonkey.net/).
2. Создайте новый userscript.
3. Скопируйте в него содержимое [`../../mangabuff_helper.js`](../../mangabuff_helper.js).
4. Сохраните скрипт.
5. Откройте `mangabuff.ru` и используйте кнопку `MB` в левом нижнем углу.

## Настройки

В скрипте есть отдельные модальные окна для:

- автоскролла;
- автоквиза;
- автокомментов.

Каждое модальное окно поддерживает внутреннюю прокрутку, а также кнопки прокрутки на случай, если колесо мыши в браузере не срабатывает.

## Примечания

- Скрипт зависит от текущей структуры DOM и поведения запросов на `mangabuff.ru`.
- Если сайт изменит классы, футер ридера или поведение endpoint-ов, часть функций может потребовать обновления.
- Комментарии всё ещё могут блокироваться антиспамом сайта.
- Автоматизация действий на сайте может нарушать его правила. Используйте скрипт на свой риск.
